# Proceso de onboarding cliente
Quiero crear un proyecto front end sencillo usando angular 19. Aplicaremos una arquitectura limpia.
Hagamoslo enfocandonos en la responsividad, manejemos en scss todas esas configuraciones de diseño para centralizar diseños y reutilizabilidad.

## Pantalla de bienvenida
El onboarding va a consistir en 3 pasos. Iniciaremos explicando que este proceso requerira 3 sencillos pasos para poder brindarte un credito.

## Paso 1
En el paso 1 ingresaremos informacion de contacto e identidad.
Seria algo como:

Correo ________________
